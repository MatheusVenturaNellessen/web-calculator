<h1>Calculadora Web</h1>
<p>Esse projeto é uma calculadora web desenvolvida em HTML, CSS e JavaScript. A mesma realiza cálculos matemáticos básicos via entrada de dadaos por usuário.</p>

<h2>Tecnologias utilizadas</h2>
<ul>
    <li>HTML;
    <li>CSS;
    <li>JavaScript;
    <li>Visual Studio Code - Versão: 1.98.2.
</ul>

<h2>Estruturas das pastas do projeto</h2>

    web-calculator/
    │
    ├── assets/                 
    │   ├── css/                    # Arquivos de estilo da calculadora
    │   │   └── style.css       
    │   ├── js/                     # Scripts da calculadora
    │   │   └── main.js         
    │   └── img/                    # Imagens utilizadas na calculadora
    │       └── delete-icon.png 
    │
    └── index.html                  # Página principal da calculadora

<h2>Como rodar o projeto</h2>

<p>Não há necessiadade de instalar nenhuma depedência para rodar o projeto, apenas siga o passo-a-passo a seguir:</p>

<ul>
    <li>Clone o repositório:

    git clone https://github.com/MatheusVenturaNellessen/web-calculator.git

</ul>
<ul>
    <li>Navegue até o diretório do projeto:

    cd web-calculator

</ul>
<ul>
    <li>Abra o arquivo index.html em seu navegador para visualizar a calculadora. 
</ul>

<h2>Funcionalidades</h2>
<ul>
    <li>A página exibirá uma interface de calculadora com botões numéricos e de operações matemáticas;
    <li>Ao clicar nos botões, o usuário poderá realizar cálculos matemáticos básicos, inclusive expressões matemáticas:
        <ul>
            <li>Soma ( + );
            <li>Subtração ( - );
            <li>Multiplicação ( * );
            <li>Divisão ( / );
            <li>Exponenciação ( ** ).
        </ul>
    <li>O botão ( = ) exibe o resultado da operação;
    <li>O botão ( C ) limpa o display e renicia o cálculo;
    <li>O botão ( « ) apaga a última entrada de caracter. 
    <li>Se o cálculo não for possível realizar, um alerta aparecerá informando o usuário;
</ul>
