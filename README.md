<h1>Calculadora Web</h1>
<p>Esse projeto é uma calculadora desenvolvida em HTML, CSS e JavaScript. A mesma realiza cálculos matemáticos básicos, como soma, subtração, entre outros.</p>

<h2>Tecnologias utilizadas</h2>
<ul>
    <li>HTML;
    <li>CSS;
    <li>JavaScript;
    <li>Visual Studio Code - Versão: 1.98.2
</ul>

<h2>Estruturas das pastas / arquivos do projeto</h2>

``` Bash
web-calculator/
│
├── assets/                 
│   ├── css/                
│   │   └── style.css       
│   ├── js/                 
│   │   └── main.js         
│   └── img/                
│       └── delete-icon.png 
│
└── index.html              
```

<h2>Como rodar o projeto</h2>
<ul>
    <li>Clone o repositório:
    ``` Bash
    git clone https://github.com/MatheusVenturaNellessen/web-calculator.git
    ```
    <li>Navegue até o diretório do projeto:
    ``` Bash
    cd web-calculator
    ```
    <li>Abra o arquivo index.html em seu navegador para visualizar a calculadora. 
</ul>

<h2>Funcionalidades</h2>
<ol>
    <li>A página exibirá uma interface de calculadora com botões numéricos e de operações matemáticas;
    <li>Ao clicar nos botões, o usuário poderá realizar cálculos matemáticos;
    <li>O botão "=" exibe o resultado da operação;
    <li>O botão "C" limpa o display e renecia o cálculo;
    <li>O botão "«" apaga o último operando clicado.
    <li>Se o cálculo não for possível realizar, um pop-up aparecerá informando o usuário;
</ol>
